# CAPE Pure Mathematics Unit 2 (2026) – Paper 2 Solutions

This repository contains beautifully typeset solutions for the **CAPE Pure Mathematics Unit 2 2026 Paper 2 exam**, designed using LaTeX to serve as a resource for upcoming students taking this examination. 

---

## Features

* **Instant PDF Viewing:** Includes a pre-compiled, production-ready PDF (`.pdf`) so you can view the solutions instantly without needing a local LaTeX compiler.
* **Modular Codebase:** The LaTeX source is split into logical modules (preamble, font configuration, and main content) to keep the repository organized, maintainable and easy to modify for future projects.
* **Typesetting & Formatting:** Leverages the extensive LaTeX library and additional packages to cleanly display mathematical equations, proofs and matrices. 

---

## Repository Structure

* **`Pure_Mathematics_U2_2026_P2_Solutions.tex`** — The primary LaTeX file containing the written solutions.
* **`Pure_Mathematics_U2_2026_P2_Solutions.pdf`** — The final compiled document, optimized for reading and printing.
* **`Preamble.tex`** — Centralized document configuration, custom packages, and macro definitions.
* **`letterfonts.tex`** — Font configurations and custom mathematical character mappings.
